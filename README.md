# Conversor de Unidades de Comprimento

Projeto desenvolvido para realizar conversões entre diferentes unidades de comprimento, como metros, quilômetros, centímetros, milímetros, milhas, polegadas, pés e jardas.

## Como executar

1. Clone o repositório:

   ```bash
   git clone https://github.com/pedrolucasso832/conversor-unidades-comprimento.git
   ```

2. Acesse a pasta do projeto:

   ```bash
   cd conversor-unidades-comprimento
   ```

3. Abra o arquivo `index.html` em um navegador.

O projeto é executado diretamente no navegador e não requer a instalação de dependências.
